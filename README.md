# WIP 

## Technologies Used:
* Next.js
* TypeScript
* Tailwind
* Rough.js

## Currently you are able to:
* Draw lines
* Switch between drawing tools
* Pan the canvas using middle mouse
* Zoom in/out the canvas relative to the mouse (ctrl+wheel)
* Scroll up and down the canvas
* Undo/Redo using shortcuts
* Resize the canvas
